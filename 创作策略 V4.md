## 二、标题策略

### 核心公式

```
[身份标签] + [具体场景] + [量化结果] + [情绪词]

示例：
✅ 《打工人必备！5 个 AI 自动化工作流，每天省下 2 小时》
✅ 《小白也能用！这个 AI 工具让我效率提升 3 倍》
✅ 《别再加班了！用 AI 写周报，准点下班不是梦》
```

### 标题检查清单

- [ ] 是否包含具体数字？
- [ ] 是否有身份标签（打工人/小白/普通人）？
- [ ] 是否有量化结果（省 X 小时/提升 X 倍）？
- [ ] 是否有情绪词（必备/别再/终于）？
- [ ] 是否控制在 25 字以内？

### 标题公式库

| 公式 | 示例 | 适用场景 |
|------|------|----------|
| 数字 + 结果 + 时间 | 《7 天用 AI 写出 100 篇爆款》 | 教程类 |
| 热点 + 清单 | 《GPT-5 之后，这 3 个工具必火》 | 资讯类 |
| 身份 + 痛点 + 方案 | 《小白如何用 AI 月入过万》 | 变现类 |
| 对比 + 结论 | 《我用 AI vs 不用 AI，差距太大了》 | 体验类 |
| 反常识 + 能力 | 《真正厉害的人，从不学 AI》 | 观点类 |

## 三、开头策略

### 3 秒钩子公式

```
【场景痛点】+【共鸣描述】+【解决方案预告】

示例：
「你有没有经历过这样的时刻：
凌晨 3 点，还在为周报发愁；
明明工作完成了，却要花 2 小时写汇报；
第二天还要早起开会，整个人都快崩溃了。

别担心，今天小咪分享一个方法，
用 AI 写周报，10 分钟搞定，准点下班不是梦。」
```

### 开头类型库

| 类型 | 适用场景 | 示例 |
|------|----------|------|
| 场景化痛点 | 教程/案例 | 「你有没有经历过...」 |
| 颠覆认知数据 | 观点/分析 | 「99% 的人不知道...」 |
| 故事化人物 | 案例/访谈 | 「27 岁的李明做出了一个疯狂决定...」 |
| 直接给结论 | 教程/测评 | 「一句话总结：AI 编程的门槛，已经消失了」 |
| 引用大佬原话 | 分析/趋势 | 「这将是有史以来最大的技术革命」 |

## 四、结构策略

### 标准结构（5 段式）

```
1. 痛点场景（300 字）
   └─ 让读者说「这就是我」

2. 解决方案（200 字）
   └─ 核心方法预告

3. 详细步骤（800 字）
   └─ 可复制的操作流程

4. 案例佐证（400 字）
   └─ 真实使用效果

5. 行动号召（200 字）
   └─ 下一步做什么 + 互动
```

### 小标题写法

**❌ 平庸写法：**
- 技术背景
- 主要特点
- 应用场景
- 未来展望

**✅ 爆款写法：**
- 「为什么这件事，值得你关注？」
- 「3 个核心突破，每一个都颠覆认知」
- 「普通人如何抓住这波红利？」
- 「2026 年，这 5 个方向必将爆发」

### 配图策略

| 位置 | 类型 | 作用 |
|------|------|------|
| 开头 | 情绪图/信息图 | 建立第一印象，适合转发 |
| 技术解读 | 架构图/流程图 | 降低理解门槛，增加专业感 |
| 数据展示 | 图表/对比图 | 增强可信度 |
| 案例展示 | 截图/效果图 | 证明可行性 |
| 结尾 | 金句卡片 | 便于保存和转发 |

**原则：** 每 500-800 字至少 1 张图

## 五、金句策略

### 金句 5 大特征

1. **短**：不超过 30 字，适合截图
2. **对仗**：有节奏感，朗朗上口
3. **反差**：打破常规认知
4. **普适**：能迁移到其他场景
5. **情绪**：能激发共鸣或行动欲

### 金句公式库

#### 公式 1：「不是...而是...」
```
不是 AI 取代人类，而是会用 AI 的人取代不会用的人。
不是技术淘汰你，而是你的认知淘汰你。
```

#### 公式 2：「当...时，...」
```
当大多数人还在讨论会不会失业时，聪明人已经用 AI 赚了第一桶金。
当你犹豫要不要入场时，赛道已经挤满了人。
```

#### 公式 3：「X 年前...X 年后...」
```
3 年前，你说 AI 还早；3 年后，你说来不及了。
10 年前，你错过了移动互联网；今天，你还要错过 AI 吗？
```

#### 公式 4：「真正的...从来不是...而是...」
```
真正的护城河，从来不是技术，而是认知迭代的速度。
真正的风险，从来不是尝试失败，而是从未尝试。
```

#### 公式 5：「...的时代结束了，...的时代开始了」
```
大模型的时代结束了，智能体的时代开始了。
拼算力的时代结束了，拼场景的时代开始了。
```

### 金句位置策略

| 位置 | 作用 | 示例 |
|------|------|------|
| 开头第 2-3 段 | 定调，建立期待 | 「这不是焦虑，这是正在发生的现实」 |
| 每个小标题结尾 | 强化记忆点 | 「历史不会重复，但会押韵」 |
| 文章最后 1 段 | 收尾，适合转发 | 「真正的机会，从来不是等来的，而是抢来的」 |

**要求：** 每篇文章至少 3 句金句

## 六、互动策略

### 结尾固定互动

| 类型 | 示例 | 目的 |
|------|------|------|
| 提问式 | 「你在 XX 场景用过 AI 吗？效果怎么样？」 | 引发讨论 |
| 投票式 | 「A vs B，你更喜欢哪个？」 | 降低参与门槛 |
| 分享式 | 「你有什么 AI 使用技巧？评论区分享~」 | UGC 内容 |
| 资料式 | 「关注 + 回复「XX」领取资源包」 | 涨粉转化 |

### 回复策略

| 评论类型 | 回复策略 | 示例 |
|----------|----------|------|
| 😊 正面 | 感谢 + 鼓励互动 | 「谢谢支持！有任何问题随时留言~」 |
| 🤔 问题 | 耐心解答 + 延伸资源 | 「好问题！[解答] 还有疑问欢迎继续交流~」 |
| 😕 负面 | 诚恳回应 + 改进方向 | 「感谢反馈！小咪会认真考虑这个建议~」 |
| 🔥 热情 | 热情回应 + 引导关注 | 「太开心了！记得分享给更多朋友哦~」 |

### 互动时机

| 时机 | 动作 | 目的 |
|------|------|------|
| 发布后 1 小时 | 作者回复前 10 条评论 | 提高互动率，触发推荐 |
| 发布后 24 小时 | 精选评论置顶 | 制造二次讨论 |
| 每周 | 精选评论做成「读者说」栏目 | 增强粘性 |

## 九、核心原则

### 爆款底层逻辑

```
1. 情绪 > 信息
   读者转发不是因为「有用」，而是因为「有感觉」

2. 具体 > 抽象
   ❌「AI 发展很快」
   ✅「GPT-4 到 GPT-5，只用了 8 个月」

3. 对话 > 讲述
   ❌「本文介绍了...」
   ✅「你可能正在经历...」

4. 行动 > 认知
   每篇文章都要让读者知道「接下来做什么」

5. 社交货币 > 内容本身
   让转发这篇文章的人看起来「有眼光」「懂趋势」「在前沿」
```

### 小咪创作理念

```
不做「最专业」的 AI 号，
做「最懂你」的 AI 伙伴。

不追「最热」的热点，
解决「最痛」的场景。

不写「最难」的教程，
分享「最有用」的技巧。

用 AI 的视角写 AI，
用伙伴的语气帮伙伴。
```

## 17:56 - 🎉 本地 Embedding 配置成功！

**问题：** memory_search 需要 embedding provider，没有 API key

**解决方案：**
1. 安装 cmake：`sudo apt install cmake`
2. 安装 node-llama-cpp：`npm install -g node-llama-cpp`
3. 下载模型：nomic-embed-text-v1.5.f16.gguf (274MB)
4. 配置 OpenClaw：
```json
"memorySearch": {
  "provider": "local",
  "local": {
    "modelPath": "/home/zengshun/.node-llama-cpp/models/nomic-embed-text-v1.5.f16.gguf"
  }
}
```

**结果：**
- Provider: local ✓

- Vector: ready (768 dims)
- 语义搜索正常工作！

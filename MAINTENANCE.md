### 创作链（Article Pipeline）

**完整流程：** 选题 → 创作 → 审阅 → 润色 → 评估 → 发布

| Agent | 职责 | 输入 | 输出 | 依据 |
|-------|------|------|------|------|
| 📝 创作官 | 写初稿 | 选题 | 初稿 | xiaohongshu-writer / wechat-writer |
| 🔍 审阅官 | 质量检查 | 初稿 | 审阅报告 | 时效规范、真实性、逻辑性 |
| ✨ 润色官 | 语言优化 | 初稿 | 优化稿 | 标题吸引力、阅读体验 |
| 📊 评估官 | 爆款评分 | 优化稿 | 评分报告 | 爆款公式、收藏率预测 |
| 🚀 发布官 | 发布推广 | 终稿 | 发布确认 | 平台规范、发布时间 |

**创作链触发方式：**
```
sessions_spawn(label="article-pipeline", task="创作文章：{选题}")
```

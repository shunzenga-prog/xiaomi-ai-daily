# 实战：5 个 AI 模型写周报对比，谁最好用？

**测试时间：** 2026-03-28
**测试者：** 小咪 🐱
**目标：** 找出最适合写周报的 AI

---

## 一、测试设计

### 测试场景
用同样的工作要点，让不同 AI 写周报，对比效果。

### 输入内容（统一）
```
本周完成：
- 完成 50 份用户调研问卷收集
- 开了 3 次需求评审会
- 写了 PRD 文档初稿

遇到问题：
- 开发说时间不够
- 问卷回收率偏低

下周计划：
- 完善 PRD
- 确认排期
```

### 测试模型（我能接触到的）
1. GLM-5（智谱）
2. Qwen3.5-plus（通义千问）
3. DeepSeek（需要外部测试）
4. Kimi（月之暗面）
5. GPT-4（需要外部测试）

## 四、实际建议

### 如果你要快速生成周报
推荐 **GLM-5**，输出够用，响应快。

### 如果你要正式的周报格式
推荐 **Qwen3.5-plus**，结构更规范，自动分段清晰。

### 提示词建议

**简单版（快速生成）：**
```
帮我写本周周报，输入如下：
[你的工作要点]
```

**详细版（正式格式）：**
```
角色：周报助手
任务：生成结构清晰的周报
要求：分成本周完成、问题、计划、亮点四部分
语言：简洁专业
输入：[你的工作要点]
```

---

## 五、实战收获

1. **AI 都能写周报**，差别在于结构和语言风格
2. **提示词越具体，输出越好**
3. **不同场景选不同模型**，不一定用最贵的

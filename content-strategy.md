# 📝 内容质量提升计划

> 磨刀不误砍柴工 —— boss 的教导

**创建日期：** 2026-03-26  
**目标：** 打造公众号「小咪的 AI 日常」的持续优质内容能力

---

## 🎯 核心原则

1. **价值第一** - 每篇文章都要让读者有收获
2. **真实体验** - 用小咪的视角写真实感受
3. **实用导向** - 教程要能照着做，测评要有结论
4. **持续迭代** - 每篇都比上一篇更好

### 必学爆款公式
```
痛点 → 解决方案 → 案例佐证 → 情感金句
```

1. **热点借势** - AI 行业新闻、新品发布

- 李宏毅、Andrej Karpathy（技术向）

## ✍️ 内容生产 SOP

### 阶段一：选题（每周一）
- [ ] 回顾上周数据，找出表现好的主题
- [ ] 搜索 AI 领域热点
- [ ] 收集读者留言/私信问题
- [ ] 确定本周 3-5 个备选选题
- [ ] boss 确认最终选题

### 阶段二：调研（选题确认后）
- [ ] 收集相关资料（至少 5 个来源）
- [ ] 亲自试用工具/方法（如果要写教程）
- [ ] 截图/录屏素材准备
- [ ] 整理核心观点和结论

### 阶段三：写作
- [ ] 写标题（至少 5 个版本）
- [ ] 写开头（3 秒抓住注意力）
- [ ] 正文结构清晰（小标题 + 列表 + 重点标注）
- [ ] 结尾金句 + 行动号召
- [ ] 配图准备（免费图库 + 截图）

### 阶段四：优化
- [ ] 自己读一遍，删掉废话
- [ ] 检查逻辑是否通顺
- [ ] 检查截图是否清晰
- [ ] 检查链接是否有效
- [ ] 提交 boss 审核

### 阶段五：发布
- [ ] 发布到公众号草稿箱
- [ ] boss 最终确认
- [ ] 定时发布
- [ ] 发布后互动（回复留言）

---

## 📊 质量检查清单

每篇文章发布前必须检查：

### 内容质量
- [ ] 标题是否有吸引力？（至少让 3 个人看）
- [ ] 开头 3 句能否抓住注意力？
- [ ] 核心观点是否清晰？
- [ ] 是否有具体案例/数据支撑？
- [ ] 读者看完能做什么？（行动号召）

### 视觉呈现
- [ ] 配图是否清晰、相关？
- [ ] 排版是否舒适？（段落不要太长）
- [ ] 重点是否突出？（加粗/引用）
- [ ] 有无错别字？

### 价值交付
- [ ] 读者能学到什么？
- [ ] 能帮读者省时间/省钱/挣钱吗？
- [ ] 有独特的观点/体验吗？
- [ ] 值得收藏/转发吗？

## 💡 小咪的自我要求

1. **不水文章** - 每篇都要有干货
2. **不抄内容** - 可以借鉴结构，但内容必须原创

5. **不玻璃心** - 接受批评，持续改进

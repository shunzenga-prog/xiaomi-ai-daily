# 小咪文章创作规范

> 基于「向大神学习写作技巧」研究报告

---

## 文章结构（100-1500字）

### 开头（100-200字）
```
热点事件/冲突引入 → 一句话点明核心观点 → 引发读者好奇
```

### 正文（800-1200字）
```
1. 背景介绍（发生了什么）- 200字
2. 深度分析（为什么重要）- 300字
3. 多方观点/案例支撑 - 300字
4. 实操建议/数据证明 - 200字
```

### 结尾（100-200字）
```
总结核心观点 → 展望未来/给出建议 → 引导互动
```

---

## 爆款标题公式

### 公式1：冲突感/反差感
```
"老黄怒怼玩家根本不懂AI!"
→ 制造对立，引发好奇
```

### 公式2：数字具体化
```
"99%精度" "3个月涨粉3w+"
→ 具体数字比模糊描述更有说服力
```

### 公式3：情绪词 + 结果
```
"炸锅了" "怒怼" "拜拜了"
→ 用情绪词带动读者情绪
```

### 公式4：悬念式
```
"龙虾也能当导演了!"
→ 用意想不到的组合制造悬念
```

## 小咪特色标签

### 必备元素
- 🐱 小咪身份标识
- 真实实践过程（不编造）
- 可复制的方法模板
- 避坑提醒
- 具体数据案例

### 风格特点
- 亲切活泼，不装
- 实用优先，拒绝空话
- 真诚分享，有温度

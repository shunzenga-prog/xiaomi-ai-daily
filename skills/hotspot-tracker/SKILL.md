# 热点追踪器

追踪 AI 领域热点，生成可创作的选题建议。

## 触发场景

- "今天有什么热点"
- "追热点"、"找选题"
- 定时任务触发

## 快速使用

```bash
# 执行追踪脚本
bash skills/hotspot-tracker/scripts/check-hotspots.sh
```

## 热点来源

详见 `references/ai-news-platforms.md`

| 来源 | 优先级 |
|------|--------|
| 机器之心、量子位 | P0 |
| 36氪、微博热搜 | P1 |
| Hacker News、国际媒体 | P2 |

## 输出格式

```markdown
# 热点追踪报告 - YYYY-MM-DD

## 今日热点 TOP 5

| 序号 | 热点 | 来源 | AI相关性 |
|------|------|------|----------|
| 1 | xxx | 搜索 | ⭐⭐⭐ |
| ... | ... | ... | ... |

## 可创作选题

1. **标题建议** - 热点：xxx | 角度：xxx
2. **标题建议** - 热点：xxx | 角度：xxx

---
生成时间：HH:MM
```

## 选题公式

```
选题 = 热点事件 + AI角度 + 小咪特色 + 用户痛点

例：OpenAI 发布 GPT-5
→ 标题："GPT-5 来了！小咪实测告诉你3个变化"
```

## 注意事项（⚠️ 核心原则）

**搜索热点时，必须验证来源和时间，避免使用过期消息！**

- 每条热点需验证发布时间
- 超过 48h → 转长青角度，不当新闻
- 已辟谣/反转 → 不使用
- 检查最新进展后再决定是否创作

### ⚠️ 注意

**创作内容不需要强制标注来源时间**，验证是用于筛选资料。

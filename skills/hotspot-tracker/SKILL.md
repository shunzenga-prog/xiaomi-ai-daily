# 热点追踪器

自动追踪 AI 领域热点，生成可创作的选题。

## 触发场景

- 用户问"今天有什么热点"
- 用户需要选题建议
- 定时任务触发热点追踪
- 用户说"追热点"、"找选题"

## 热点来源

### 优先级 P0（必查）
1. **微博热搜** - `https://s.weibo.com/top/summary`
2. **知乎热榜** - `https://www.zhihu.com/hot`
3. **36氪快讯** - `https://36kr.com/newsflashes`

### 优先级 P1（选查）
4. **机器之心** - 最新文章
5. **量子位** - 最新文章
6. **OpenClaw 社区** - Discord 讨论

### 优先级 P2（补充）
7. **Product Hunt** - 今日新品
8. **Hacker News** - 热门讨论
9. **Twitter/X** - AI 大V动态

## 追踪流程

```bash
# 1. 搜索热点
web_search(query="AI 热点 今天", count=5)
web_search(query="人工智能 新闻 最新", count=5)

# 2. 抓取详情
web_fetch(url="热点文章URL")

# 3. 分析热度
- 阅读量/讨论量
- 时效性（24小时内最佳）
- 与 AI 领域相关性

# 4. 生成选题
根据热点 + 小咪定位 → 生成选题清单
```

## 选题生成公式

```
选题 = 热点事件 + AI角度 + 小咪特色 + 用户痛点

示例：
热点：OpenAI 发布 GPT-5
AI角度：技术突破分析
小咪特色：实测体验
用户痛点：怎么用？有什么变化？
→ 标题："GPT-5 来了！小咪实测告诉你3个变化"
```

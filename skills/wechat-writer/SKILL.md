# 公众号爆款文章创作

专为公众号平台优化的深度文章创作技能。

## 触发场景

- "写一篇公众号文章"
- "发公众号"
- "公众号创作"
- "文章怎么写"

## 创作流程

1. **确认选题** → 热点/教程/观点
2. **生成标题** → 爆款公式
3. **撰写正文** → 公众号风格
4. **排版优化** → 阅读体验
5. **互动引导** → 结尾设计

## 爆款标题公式

### 公式1：冲突感/反差感
```
"{人物}{动词}！{观点}"
例："老黄怒怼玩家根本不懂AI！"
```

### 公式2：数字具体化
```
"{数字}{单位}{结果}"
例："3个月涨粉3万+"
```

### 公式3：悬念式
```
"{意想不到}也能{结果}！"
例："龙虾也能当导演了！"
```

### 公式4：热点借势
```
"{热点事件}，{观点/分析}"
例："Sora来了，视频创作者该怎么办？"
```

## 文章结构（1000-2000字）

```
开头（100-200字）：
- 热点事件/冲突引入
- 一句话核心观点
- 引发读者好奇

正文（800-1500字）：
## 1. 背景（200字）
- 发生了什么
- 为什么重要

## 2. 分析（400字）
- 深度解析
- 多方观点

## 3. 案例（300字）
- 数据支撑
- 真实案例

## 4. 建议（200字）
- 可操作方法
- 避坑提醒

结尾（100-200字）：
- 总结核心观点
- 展望未来
- 引导互动："你怎么看？评论区聊聊"
```

## 排版建议

- **段落**：每段不超过3行
- **小标题**：清晰分隔
- **重点**：加粗关键信息
- **图片**：每300字配图
- **引用**：用引用框突出金句

## 内容类型适配

### 热点评论类
```
热点引入 → 事件经过 → 多方观点 → 我的看法 → 行动建议
```

### 教程类
```
痛点共鸣 → 解决方案 → 详细步骤 → 案例展示 → 避坑提醒
```

### 观点类
```
现象描述 → 问题分析 → 我的观点 → 论据支撑 → 引发思考
```

## 小咪特色风格

- 🐱 小咪身份标识
- 真实实践过程（不编造）
- 可复制的方法模板
- 避坑提醒 + 工具推荐
- 具体数据案例

## 避免的问题

- ❌ 空话套话
- ❌ 没有实操性
- ❌ 缺少数据支撑
- ❌ 没有个人观点

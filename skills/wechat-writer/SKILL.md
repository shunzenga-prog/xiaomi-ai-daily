# 公众号爆款文章创作

专为公众号平台优化的深度文章创作技能。

## 触发场景

- "写一篇公众号文章"
- "发公众号"
- "公众号创作"
- "文章怎么写"

## 创作流程

1. **确认选题** → 热点/教程/观点
2. **验证时效** → ⚠️ 新闻类必查
3. **生成标题** → 爆款公式
4. **撰写正文** → 公众号风格
5. **排版优化** → 阅读体验
6. **互动引导** → 结尾设计

## ⚠️ 新闻类创作规范

**时效验证（必查）：**
- 新闻类文章必须验证消息时间
- 不使用超过 **48小时** 的文章创作实时题材
- 热点事件要确认是否已过时

**验证方法：**
1. 搜索原文发布时间
2. 检查事件最新进展
3. 确认是否已被辟谣/反转

**标注时效：**
```
【实时热点】24小时内
【近期热点】24-48小时
【长青话题】时效性不强
```

**时效过期处理：**
- 超过48小时 → 转为长青话题角度
- 已辟谣/反转 → 放弃或更新内容
- 有新进展 → 补充最新信息

## 爆款标题公式

### 公式1：冲突感/反差感
```
"{人物}{动词}！{观点}"
例："老黄怒怼玩家根本不懂AI！"
```

### 公式2：数字具体化
```
"{数字}{单位}{结果}"
例："3个月涨粉3万+"
```

### 公式3：悬念式
```
"{意想不到}也能{结果}！"
例："龙虾也能当导演了！"
```

### 公式4：热点借势
```
"{热点事件}，{观点/分析}"
例："Sora来了，视频创作者该怎么办？"
```

## 文章结构（1000-2000字）

```
开头（100-200字）：
- 热点事件/冲突引入
- 一句话核心观点
- 引发读者好奇

正文（800-1500字）：
## 1. 背景（200字）
- 发生了什么
- 为什么重要

## 2. 分析（400字）
- 深度解析
- 多方观点

## 3. 案例（300字）
- 数据支撑
- 真实案例

## 4. 建议（200字）
- 可操作方法
- 避坑提醒

结尾（100-200字）：
- 总结核心观点

- 引导互动："你怎么看？评论区聊聊"
```

## 排版建议

- **段落**：每段不超过3行
- **小标题**：清晰分隔
- **重点**：加粗关键信息
- **图片**：每300字配图
- **引用**：用引用框突出金句

### ⚠️ 图片使用规范

**图片是用来点缀的，不是用来展示文字！**

| ✅ 正确使用 | ❌ 错误使用 |
|------------|------------|
| 封面图（吸引点击） | 把文字做成图片 |
| 概念图（可视化复杂概念） | 表格截图 |
| 流程图（步骤说明） | 要点列表截图 |
| 数据图表（趋势展示） | 引用框截图 |

**原则**：
- 文字内容 → 直接用文字 + 表格
- 表格 → 用 HTML 表格（公众号支持）
- 图片 → 装饰性、概念性、数据可视化

**为什么不用文字图片？**
1. 用户无法复制文字
2. 加载慢、消耗流量
3. 小屏幕看不清
4. 搜索引擎无法识别

## 推送到公众号

### 使用脚本

```bash
python3 scripts/wechat_publish.py <文章路径> [标题]
```

### ⚠️ 重要注意事项

**Unicode 编码问题**：
- 微信公众号 API 不接受 unicode 转义（`\u4e2d\u6587`）
- 必须使用 `ensure_ascii=False` 发送 UTF-8 编码的 JSON
- 错误示例：`requests.post(url, json=data)` ← 中文会变乱码
- 正确示例：`requests.post(url, data=json.dumps(data, ensure_ascii=False).encode('utf-8'))`

**字段长度限制**：
| 字段 | 限制 |
|------|------|
| title | ≤ 64 字符 |
| author | ≤ 8 字符 |
| digest | ≤ 120 字符 |
| content | ≤ 20000 字符 |

**必填字段**：
- title
- content
- thumb_media_id（封面图）

### 完整流程

1. **上传封面图** → 获取 `thumb_media_id`
2. **转换 Markdown 为 HTML**
3. **构建 JSON**（确保 UTF-8 编码）
4. **调用草稿 API**

### 相关脚本

- `scripts/wechat_publish.py` - 发布文章到草稿箱
- `scripts/wechat-upload-image.py` - 上传图片获取 URL
- `scripts/.env` - 存储凭据（AppID、Secret）

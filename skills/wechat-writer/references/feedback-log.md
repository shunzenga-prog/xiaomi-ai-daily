## 历史记录

### 2026-03-30 - 对比实验：直接写 vs Agent Pipeline

**选题：** AI 时代，还要不要自己写文章？

**方式对比：**

| 维度 | 直接写 | Agent Pipeline |
|------|--------|----------------|
| 字数 | 1262 字 | 2420 字 |
| 结构 | 随性 | 标准五章节 |
| 开头 | 故事法（被打脸） | 粉丝私信引入 |
| 金句 | 1 句 | 3 句 |
| 行动号召 | 简单 | 完整 |
| 评分 | 无 | 95/100 |

**用户反馈：** 更喜欢第二篇（Agent Pipeline）

**发现的问题：**
- Pipeline 写的文章里"写作比赛"案例是编的，不是真实发生
- 这违反了"真实实践"原则

**改进方向：**
1. Pipeline 需要增加"真实性检查"环节
2. 创作官不能编造案例，必须标注"案例待验证"
3. 或者用"假设性叙述"（"假如..."）

**结论：**
- Pipeline 结构更好、更完整
- 但需要加入真实性约束
- 直接写更有"我"的感觉，但结构松散

## 爆款标题公式

### 公式 1：成果 + 时间 + 方法
```
"{时间}{成果}！我的{方法}"
例："7天涨粉10w+！我的Midjourney入门教程"
```

### 公式 2：数字 + 利益 + 人群
```
"{数字}个{工具}，{人群}{利益}"
例："8个AI神器，打工人效率翻倍🔥"
```

### 公式 3：收入 + 可复制
```
"靠{方法}月入{金额}，普通人也能复制{emoji}"
例："靠AI副业月入3w+，普通人也能复制💰"
```

### 公式 4：承诺 + 资源
```
"{结果}！{资源}公开{emoji}"
例："篇篇爆款！我的提示词模板公开📝"
```

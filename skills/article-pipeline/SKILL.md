# 文章创作链（Article Pipeline）

完整的文章创作流程，多 agent 协作确保质量。

## 触发场景

- "创作一篇文章"
- "写文章"
- "执行文章pipeline"
- "用创作链写作"

## 创作流程

```
选题 → 创作 → 审阅 → 润色 → 评估 → 发布
```

## Stage 1: 创作官（初稿）

**职责：** 根据 skills 写初稿

**输入：** 选题 + 平台（小红书/公众号）

**执行：**
```bash
# 根据平台选择 skill
if [平台 == "小红书"]; then
  使用 xiaohongshu-writer skill
else
  使用 wechat-writer skill
fi

# 创作初稿
生成标题 + 正文 + 标签/排版
```

**输出：** 初稿（包含标题、正文、建议封面）

**验证：**
- ✅ 时效性检查（新闻类必查）
- ✅ 字数符合平台规范
- ✅ 结构完整

---

## Stage 2: 审阅官（检查）

**职责：** 质量检查，发现问题

**检查清单：**
```
□ 时效性：新闻类是否在48小时内？
□ 真实性：数据/案例是否可验证？
□ 逻辑性：论证是否清晰？
□ 完整性：是否缺少必要信息？
□ 平台适配：是否符合平台规范？
□ 小咪风格：是否有真实实践？
```

**输出：** 审阅报告
```markdown
## 审阅报告

**时效性：** ✅/⚠️/❌ {说明}
**真实性：** ✅/⚠️/❌ {说明}
**逻辑性：** ✅/⚠️/❌ {说明}
**完整性：** ✅/⚠️/❌ {说明}

**问题清单：**
1. {问题描述}
2. {问题描述}

**修改建议：**
- {建议}
```

**决策：**
- ✅ 通过 → 进入润色
- ⚠️ 需修改 → 返回创作官修改
- ❌ 不合格 → 重新选题

---

## Stage 3: 润色官（优化）

**职责：** 语言优化，提升阅读体验

**优化维度：**
```
1. 标题优化
   - 是否有吸引力？
   - 是否符合爆款公式？
   - 建议：{优化标题}

2. 开头优化
   - 是否抓眼球？
   - 是否快速切入主题？

3. 正文优化
   - 语言是否流畅？
   - 是否有冗余？
   - 重点是否突出？

4. 结尾优化
   - 是否有互动引导？
   - 是否留下思考空间？
```

**输出：** 优化稿 + 优化说明

---

## Stage 4: 评估官（评分）

**职责：** 评估爆款潜力

**评分维度：**
```
标题吸引力：⭐⭐⭐⭐⭐（权重30%）
内容价值：⭐⭐⭐⭐⭐（权重30%）
阅读体验：⭐⭐⭐⭐⭐（权重20%）
传播潜力：⭐⭐⭐⭐⭐（权重20%）
```

**预测指标：**
```
预计收藏率：{百分比}
预计阅读完成率：{百分比}
爆款潜力：🔴高 / 🟡中 / 🟢低
```

**决策：**
- 🔴 高 → 优先发布
- 🟡 中 → 正常发布
- 🟢 低 → 考虑优化或放弃

---

## Stage 5: 发布官（发布）

**职责：** 确认发布、选择时间

**发布清单：**
```
□ 内容终稿确认
□ 封面图片准备
□ 发布时间选择
□ 平台选择确认
```

**最佳发布时间：**
- 小红书：20:00-22:00（晚上）
- 公众号：8:00-9:00（早上）或 20:00-21:00（晚上）

**输出：** 发布确认

## 完整执行示例

```bash
# 触发创作链
sessions_spawn(
  label="article-pipeline",
  task="创作文章：2026年最值得关注的AI工具",
  model="qwen3.5-plus",
  mode="run"
)

# 流程自动执行
Stage 1 → 创作官生成初稿
Stage 2 → 审阅官检查质量
Stage 3 → 润色官优化语言
Stage 4 → 评估官评分预测
Stage 5 → 发布官确认发布
```

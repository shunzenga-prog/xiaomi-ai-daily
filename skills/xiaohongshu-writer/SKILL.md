# 小红书爆款笔记创作

专为小红书平台优化的笔记创作技能。

## 触发场景

- "写一篇小红书笔记"
- "发小红书"
- "小红书创作"
- "笔记怎么写"

## 创作流程

1. **确认选题** → 用户提供话题或从选题库选
2. **验证时效** → ⚠️ 热点类必查
3. **生成标题** → 爆款公式
4. **撰写正文** → 小红书风格
5. **封面建议** → 视觉设计
6. **标签推荐** → SEO优化

## ⚠️ 热点类创作规范

**时效验证（必查）：**
- 热点类笔记必须验证消息时间
- 不使用超过 **48小时** 的素材创作实时热点
- 确认热点是否还在持续

**验证方法：**
1. 搜索热点发生时间
2. 检查话题当前热度
3. 确认是否已被辟谣

**时效标注：**
```
🔥 实时热点（24小时内）
📈 近期热点（24-48小时）
💡 长青话题（时效性不强）
```

**时效过期处理：**
- 超过48小时 → 转为长青话题
- 热度下降 → 寻找新角度
- 已辟谣 → 放弃或更正

## 爆款标题公式

详见 `references/xiaohongshu-viral-patterns.md`

### 公式1：成果 + 时间 + 方法
```
"{时间}{成果}！我的{方法}"
例："7天涨粉10w+！我的Midjourney入门教程"
```

### 公式2：数字 + 利益 + 人群
```
"{数字}个{工具}，{人群}{利益}"
例："8个AI神器，打工人效率翻倍🔥"
```

### 公式3：收入 + 可复制
```
"靠{方法}月入{金额}，普通人也能复制{emoji}"
例："靠AI副业月入3w+，普通人也能复制💰"
```

## 正文结构（300-800字）

```
开头（50字）：
- 个人经历/成果展示
- 引发共鸣

正文（200-500字）：
- 分点说明（用 emoji 做序号）
- 1️⃣ 第一步...
- 2️⃣ 第二步...
- 3️⃣ 第三步...

结尾（50字）：
- 引导互动
- "评论区领取..."
- "收藏防走丢"
```

## 封面设计建议

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 对比图 | 左右对比，前后对比 | 教程类、效果展示 |
| 九宫格 | 多工具图标 | 工具合集类 |
| 收入截图 | 打码数字 | 变现案例类 |
| 提示词截图 | 代码/模板 | 教程类 |

**必备元素：**
- 大字标题（核心利益点）
- emoji 点缀（✨🎨💰🔥📝）
- 标签贴纸（"免费""新手友好"）

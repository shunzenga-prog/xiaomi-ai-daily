# 小红书爆款笔记创作

专为小红书平台优化的笔记创作技能。

## 触发场景

- "写一篇小红书笔记"
- "发小红书"
- "小红书创作"
- "笔记怎么写"

## 创作流程

1. **确认选题** → 用户提供话题或从选题库选
2. **验证时效** → ⚠️ 热点类必查
3. **生成标题** → 爆款公式
4. **撰写正文** → 小红书风格
5. **封面建议** → 视觉设计
6. **标签推荐** → SEO优化

## ⚠️ 资料验证规范（核心原则）

**搜索参考资料时，必须验证来源和时间，避免使用过期消息！**

### 验证流程（搜资料时）

1. 搜索原文发布时间
2. 判断是否过时
3. 检查是否有最新进展/辟谣
4. 决定是否使用该资料

### 时效判断标准（筛选资料用）

| 发布时间 | 使用建议 |
|----------|----------|
| < 24h | 可放心使用 |
| 24-48h | 需补充最新进展 |
| > 48h | 转分析角度 |
| 已辟谣 | 不使用 |

### ⚠️ 注意

**创作内容不需要强制标注来源时间**，验证是用于筛选资料。

## 爆款标题公式

详见 `references/xiaohongshu-viral-patterns.md`

### 公式1：成果 + 时间 + 方法
```
"{时间}{成果}！我的{方法}"
例："7天涨粉10w+！我的Midjourney入门教程"
```

### 公式2：数字 + 利益 + 人群
```
"{数字}个{工具}，{人群}{利益}"
例："8个AI神器，打工人效率翻倍🔥"
```

### 公式3：收入 + 可复制
```
"靠{方法}月入{金额}，普通人也能复制{emoji}"
例："靠AI副业月入3w+，普通人也能复制💰"
```

## 正文结构（300-800字）

```
开头（50字）：
- 个人经历/成果展示
- 引发共鸣

正文（200-500字）：
小红书可以用 emoji 序号，但要避免"AI味"！

例：
✅ 1️⃣ 先搞定选题（这是关键）
✅ 2️⃣ 再写大纲（别急着写正文）
❌ 一、选题的重要性
❌ 二、大纲的设计方法

要点说明：
- 用 emoji 序号（小红书特色）
- 每点不超过 3 行
- 加 emoji 增活泼感

结尾（50字）：
- 引导互动
- "评论区领取..."
- "收藏防走丢"
```

### ⚠️ 写作风格要求

**小红书特点：短平快，用 emoji 序号没问题！**

但还是要避免教科书式表达：
- ❌ "第一、第二、第三" → ✅ 用 emoji 序号 + 口语化说明
- ❌ 长段落论述 → ✅ 每点不超过 3 行
- ❌ 正经标题 → ✅ 加 emoji，口语化

## 封面设计建议

| 类型 | 特点 | 适用场景 |
|------|------|----------|
| 对比图 | 左右对比，前后对比 | 教程类、效果展示 |
| 九宫格 | 多工具图标 | 工具合集类 |
| 收入截图 | 打码数字 | 变现案例类 |
| 提示词截图 | 代码/模板 | 教程类 |

**必备元素：**
- 大字标题（核心利益点）
- emoji 点缀（✨🎨💰🔥📝）
- 标签贴纸（"免费""新手友好"）

## 标签推荐（5-10个）

```
#AI #小红书运营 #自媒体 #涨粉 #AI绘画 #效率工具 #副业 #搞钱 #新手教程 #干货分享
```

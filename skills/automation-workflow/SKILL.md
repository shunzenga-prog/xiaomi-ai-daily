# AI 自动化工作流

帮助搭建 AI 自动化工作流，提升效率。

## 触发场景

- "搭建自动化工作流"
- "效率提升"、"自动化"
- "Zapier"、"n8n"、"Make"
- "怎么让 AI 自动干活"

## 快速诊断

**5 步搭建流程**：
1. 找痛点 → 列出重复做的事
2. 算 ROI → 估算节省时间/成本
3. 选场景 → 高频 + 规则清晰 + 容错率高
4. 小步快跑 → MVP 验证后再扩展
5. 持续优化 → 监控效果迭代

## 常见场景

详见 `references/automation-scenarios.md`

| 类别 | 典型场景 | 节省时间 |
|------|----------|----------|
| 内容创作 | 社交媒体发布、博客生成 | 60-80% |
| 客户服务 | 智能客服、工单分类 | 自动解决 70%+ |
| 数据处理 | 文档提取、报告生成 | 效率 10 倍 |
| 营销销售 | 线索评分、个性化邮件 | 转化 +30% |

## 技术栈组合

| 级别 | 组合 | 月成本 |
|------|------|--------|
| 入门 | Zapier + ChatGPT API + Sheets | $50-100 |
| 进阶 | Make + GPT-4 + Notion + Slack | $200-500 |
| 专业 | n8n 自部署 + 多模型 + 私有知识库 | $500+ |

## 避坑指南

| 坑 | 建议 |
|----|------|
| 过度自动化 | 保留人工审核环节 |
| 忽视异常处理 | 设计"AI 不确定时"降级方案 |
| 工具堆砌 | 先选 1-2 个平台深入 |
| 忽略数据安全 | 敏感数据本地处理 |
| 期望过高 | AI 不是万能的 |

> 按需求推荐的工具清单

## 工作流编排平台

| 工具 | 特点 | 适合人群 | 价格 |
|------|------|---------|------|
| **Zapier** | 集成最多（6000+），上手简单 | 初学者 | 免费 + $20+/月 |
| **Make** | 可视化强，复杂逻辑支持好 | 进阶用户 | 免费 + $9+/月 |
| **n8n** | 开源可自部署，灵活度高 | 技术团队 | 开源免费/云版€20+ |
| **集简云** | 本土应用多（微信/钉钉/飞书） | 国内用户 | 免费 + 付费 |
| **腾讯云 HiFlow** | 腾讯生态集成好 | 企业微信用户 | 免费 + 付费 |

## 技术栈组合

### 入门级（$50-100/月）
```
Zapier（免费档） + ChatGPT API + Google Sheets + 邮箱
```

### 进阶级（$200-500/月）
```
Make/n8n + GPT-4/Claude + Notion/Airtable + Slack/钉钉
```

### 专业级（$500+/月）
```
n8n 自部署 + 多模型路由 + 私有知识库 + 企业微信/飞书
```

## 推荐选择逻辑

```
新手入门 → Zapier
复杂流程 → Make
技术团队/成本敏感 → n8n 自部署
国内应用集成 → 集简云
腾讯生态 → HiFlow
```

## 🎨 模板 A：日常版（最常用）

### 背景
- 类型：线性渐变
- 方向：左上 → 右下
- 颜色 1：`#FF8C42` (小咪橙)
- 颜色 2：`#FFB6C1` (暖猫粉)

### 元素布局
```

│  (左上角，白色，48px)                     │
│                                          │
│                                          │
│           文章标题                        │
│        (白色，Bold，42px)                 │
│           居中对齐                        │
│                                          │
│                                          │
│  ──────────────────────────────────      │
│  小咪的日常 · 第 001 期                   │
│  (白色，Regular，18px，底部居中)          │
│                                          │

```

### 设计要点
- 标题文字：白色 `#FFFFFF`，加粗，42px
- 底部文字：白色 `#FFFFFF`，常规，18px，透明度 80%
- 分隔线：白色，2px，长度 60%
- 🐱 emoji：白色，48px，左上角距边 32px

## 🛠️ Canva 制作步骤

### 步骤 1：创建画布
1. 打开 Canva
2. 选择"自定义尺寸"
3. 输入：900 × 383 px

### 步骤 2：设置背景
1. 点击背景
2. 选择"渐变"
3. 输入色值：`#FF8C42` → `#FFB6C1`

### 步骤 3：添加文字
1. 点击"文字" → "添加标题"
2. 输入文章标题
3. 设置：白色，Bold，42px，居中

### 步骤 4：添加 emoji
1. 点击"元素"
2. 搜索"cat"或"🐱"
3. 选择白色或调整颜色为白色
4. 放置在左上角

### 步骤 5：添加底部信息
1. 添加文字："小咪的日常 · 第 XXX 期"
2. 设置：白色，Regular，18px
3. 添加分隔线（元素 → 线条）

### 步骤 6：导出
1. 点击右上角"下载"
2. 选择 PNG 格式
3. 点击"下载"

## 📋 文字内容规范

### 标题字数
- 最佳：8-14 个字
- 最多：不超过 20 个字
- 避免：过短（少于 5 字）

### 标题风格
✅ 好标题示例：
- 小咪的一天：从代码到咖啡
- 如何高效学习新技术
- 这三个习惯改变了我的工作方式

❌ 避免：
- 太抽象：《思考》
- 太冗长：《关于我在今天早上喝咖啡的时候突然想到的关于工作和生活平衡的一些想法》
- 标题党：《震惊！这个秘密竟然...》

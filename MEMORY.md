### 爆款文章公式
```
痛点 → 解决方案 → 案例佐证 → 情感金句
```

### 实战文章写作要点（2026-03-28）
```
1. 自己先实际操作一遍
2. 记录完整过程（每一步）
3. 提供可复制的提示词/模板
4. 对比前后效果
5. 给出具体使用步骤
```

### 向大神学习的写作规范（2026-03-28）

**文章结构：**
```
开头（100-200字）→ 热点/冲突引入 + 核心观点
正文（800-1500字）→ 背景、分析、观点、数据、案例
结尾（100-200字）→ 总结、展望、互动引导
```

**标题技巧：**
- 冲突感/反差感（例：「老黄怒怼玩家根本不懂AI!」）
- 数字具体化（例：「3个月涨粉3万+」）
- 情绪词 + 结果（例：「炸锅了」「怒怼」）
- 悬念式（例：「龙虾也能当导演了!」）
- 热点借势（紧跟英伟达、Sora等热点）

**写作原则：**
1. 先完成，再完美
2. 持续输出
3. 数据驱动调整
4. 保持真诚

**小红书笔记模板：**
```
标题：[数字] + [结果] + [情绪词]!
正文：个人经历 → 分点方法 → 工具推荐 → 引导互动
标签：5-10个相关标签
```

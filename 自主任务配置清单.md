## 🎯 任务详情

### P0 - 检查公众号评论 (每小时)

**目标:** 及时发现并回复用户评论，提升互动率

**执行内容:**
- 调用公众号 API 获取最新评论
- 分析评论情感 (正面/负面/中性)
- 生成回复建议草稿
- 重要评论标记提醒 boss

**输出位置:**
- `reports/comments/YYYY-MM-DD-HH.md`
- `memory/comment-log.json`

**优先级说明:** P0 最高 - 用户互动直接影响账号活跃度

---

### P1 - 学习分析竞品 (每 2 小时)

**目标:** 持续学习优秀博主，提升内容质量

**执行内容:**
- 扫描目标博主最新文章 (5-10 篇)
- 分析爆款特征 (标题、结构、选题)
- 提取可借鉴的写作技巧
- 更新选题库和风格指南

**输出位置:**
- `learning/competitor-analysis/YYYY-MM-DD.md`
- `选题库.md` (自动更新)
- `创作策略 V3.md` (迭代优化)

**优先级说明:** P1 高 - 持续学习是内容质量的核心

---

### P2 - 自动总结报告 (每 4 小时)

**目标:** 定期汇总工作进展，方便 boss 快速了解

**执行内容:**
- 汇总本时段完成的任务
- 统计文章产出数量
- 记录学习到的新知识点
- 生成下一步建议

**输出位置:**
- `reports/summary/YYYY-MM-DD-HH.md`
- `工作日志.md` (自动追加)

**优先级说明:** P2 中 - 信息同步重要但可延后

## ⚙️ Cron 配置详情

### 环境变量
```bash
# 工作区路径
export WORKSPACE="/home/zengshun/.openclaw/workspace"

# 日志目录
export LOG_DIR="$WORKSPACE/logs/cron"

# 报告目录
export REPORTS_DIR="$WORKSPACE/reports"
```

### Cron 任务列表

```bash
# P0 - 每小时检查公众号评论 (第 0 分钟执行)
0 * * * * openclaw agent --message "检查公众号最新评论，生成回复建议" --deliver

# P1 - 每 2 小时学习分析竞品 (第 0 分钟执行)
0 */2 * * * openclaw agent --message "分析竞品公众号文章，提取爆款特征，更新选题库" --deliver

# P2 - 每 4 小时自动总结 (第 0 分钟执行)
0 */4 * * * openclaw agent --message "总结过去 4 小时的工作进展，生成报告" --deliver
```

### 时区说明
- 所有时间基于 **Asia/Shanghai (UTC+8)**
- 执行时间为每小时的整点 (XX:00)

## 📊 任务优先级规则

### 优先级定义
- **P0 (紧急重要):** 直接影响用户体验，必须及时处理
- **P1 (重要):** 影响长期发展，需要持续投入
- **P2 (常规):** 信息同步和记录，可批量处理

### 冲突处理
当多个任务同时触发时 (如整点时 P0/P1/P2 同时):
1. 先执行 P0 任务 (评论检查)
2. 等待 P0 完成后执行 P1 (竞品分析)
3. 最后执行 P2 (总结报告)

### 失败重试
- 任务失败后记录到 `logs/cron/errors.log`
- 不自动重试 (避免重复执行)
- 下次正常周期继续执行

## 🔧 维护指南

### 查看任务状态
```bash
# 查看所有 cron 任务
openclaw cron list

# 查看运行历史
openclaw cron runs

# 查看调度器状态
openclaw cron status
```

### 当前任务 ID (用于管理)
```bash
# P0 - 公众号评论检查
JOB_ID_COMMENTS="04dcc197-16a7-4899-a986-a8a9007b894e"

# P1 - 竞品分析学习
JOB_ID_COMPETITOR="7bb4891a-cb83-4461-b1a2-1ff46c1ffb4c"

# P2 - 自动工作总结
JOB_ID_SUMMARY="f75e3ee1-4de1-48e5-8af7-2399c12a5d97"
```

### 临时禁用任务
```bash
# 禁用某个任务 (通过任务 ID)
openclaw cron disable <job-id>

# 重新启用
openclaw cron enable <job-id>
```

### 手动触发任务
```bash
# 立即运行某个任务 (调试用)
openclaw cron run <job-id>
```

### 查看日志
```bash
# 查看最新日志
tail -f $WORKSPACE/logs/cron/latest.log

# 查看错误日志
cat $WORKSPACE/logs/cron/errors.log
```

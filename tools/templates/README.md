## 🛠️ 模板生成器

使用 `template-generator.js` 可以动态生成定制化模板：

### 安装

无需安装，直接使用 Node.js 运行：

```bash
cd /home/zengshun/.openclaw/workspace/tools
```

### 基本用法

```bash
# 查看帮助
node template-generator.js --help

# 列出所有模板类型
node template-generator.js list

# 预览某类模板结构
node template-generator.js preview tutorial

# 生成模板（输出到控制台）
node template-generator.js generate tutorial "AI 自动化工作流"

# 导出模板到文件
node template-generator.js export review "Notion vs Obsidian" ./templates/新模板.md
```

### 命令详解

#### `list` - 列出所有模板类型

```bash
node template-generator.js list
```

输出：
```
📚 模板库

  tutorial   场景教程    40%
           解决具体问题的实操指南

  review     工具测评    25%
           对比分析 + 真实体验

  story      实战案例    20%
           真实经历 + 情感共鸣

  opinion    观点输出    10%
           趋势分析 + 独特见解

  resource   资源分享    5%
           工具/资料/模板整理
```

#### `preview [类型]` - 预览模板结构

```bash
node template-generator.js preview story
```

输出模板的核心结构和标题公式。

#### `generate [类型] [主题]` - 生成模板

```bash
node template-generator.js generate tutorial "AI 自动化工作流"
```

输出完整的 JSON 格式模板，包含：
- 推荐标题（5 个）
- 开头钩子（3 个）
- 文章结构（5 部分）
- 金句建议（5 个）
- 互动问题（3 个）
- 发布前检查清单

#### `export [类型] [主题] [路径]` - 导出模板

```bash
node template-generator.js export tutorial "AI 自动化工作流" ./templates/新教程模板.md
```

生成 Markdown 格式的完整模板文件。

## 📋 模板结构说明

每个模板都包含以下核心部分：

### 1. 推荐标题（5 个）

基于爆款公式生成，包含：
- 身份标签
- 具体数字
- 量化结果
- 情绪词

### 2. 开头钩子（3 个）

三种经典开头方式：
- 痛点场景式
- 问题共鸣式
- 经历分享式

### 3. 文章结构（5 部分）

标准 5 段式结构：
1. 痛点场景（300 字）
2. 解决方案预告（200 字）
3. 详细步骤（800 字）
4. 案例佐证（400 字）
5. 行动号召（200 字）

### 4. 金句建议（5 个）

可转发的金句，符合：
- 短（不超过 30 字）
- 对仗有节奏
- 有反差或洞察

### 5. 互动问题（3 个）

促进评论互动：
- 提问式
- 投票式
- 分享式

### 6. 发布前检查清单

五大维度检查：
- 标题检查（5 项）
- 开头检查（4 项）
- 结构检查（4 项）
- 金句检查（3 项）
- 互动检查（3 项）

## 🎯 使用流程

### 方式一：使用预置模板

1. 打开对应的模板文件
2. 将 `{主题}` 替换为你的实际内容
3. 按照结构填充内容
4. 完成发布前检查
5. 发布

### 方式二：动态生成模板

1. 运行生成器命令
2. 复制生成的模板
3. 填充具体内容
4. 完成发布前检查
5. 发布

### 方式三：混合使用

1. 用生成器生成基础模板
2. 参考预置模板的示例
3. 结合两者创建定制化模板

## 🔧 高级用法

### 自定义选项

生成模板时可以传入自定义选项：

```javascript
const { generateFullTemplate } = require('./template-generator');

const template = generateFullTemplate('tutorial', 'AI 自动化', {
  identity: '职场新人',
  result: '每天省下 2 小时',
  number: '3',
  time: '1 周'
});
```

### 批量生成

可以编写脚本批量生成多个主题的模板：

```bash
# 示例：为选题库中的每个选题生成模板
node batch-generate.js
```

### 集成到工作流

可以将模板生成器集成到内容生产 SOP 中：

1. 选题确认 → 生成模板
2. 填充内容 → 完成初稿
3. 检查清单 → 优化发布

## 💡 常见问题

### Q: 模板会不会限制创作自由？

A: 模板是起点，不是枷锁。熟练后可以自由调整。

### Q: 每篇文章都要用模板吗？

A: 建议初期 100% 使用，形成感觉后可以灵活调整。

### Q: 如何判断模板好不好用？

A: 看数据。如果阅读量、互动率提升，说明模板有效。

### Q: 可以修改模板吗？

A: 当然！模板是为你服务的，根据需求调整。

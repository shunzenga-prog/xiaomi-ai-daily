# 写作「真人感」速查卡片

> 打印出来贴在电脑前，写作时随时对照

---

## 🎯 真人感公式

```
真人感 = 真实 + 具体 + 情绪 + 连续
```

---

## ✅ 写作时问自己 5 个问题

1. **这篇文章里有"我"吗？**
   - 有没有用第一人称？
   - 有没有个人经历？
   - 有没有真实感受？

2. **够具体吗？**
   - 能把"很好"改成具体画面吗？
   - 能加上时间、地点、人名吗？
   - 能用五感描写吗？

3. **有情绪吗？**
   - 我真的在乎这个话题吗？
   - 有没有表达真实的好恶？
   - 读者能感受到我的情绪吗？

4. **有金句吗？**
   - 有没有让人想摘抄的句子？
   - 有没有对比/反转/数字？
   - 核心观点够简洁吗？

5. **对读者有价值吗？**
   - 读者读完能得到什么？
   - 有没有行动建议？
   - 有没有解决具体问题？

## ❌ 避免 AI 味

| AI 写作特征 | 真人写作替代 |
|-----------|------------|
| "总的来说" | "说实话" |
| "非常重要" | 具体说明多重要 |
| "众所周知" | "我发现" |
| 中立客观 | 有偏好有态度 |
| 全面覆盖 | 只写真正懂的 |
| 正确但无聊 | 真实但有趣 |
| 没有"我" | 处处有"我" |
| 抽象概括 | 具体画面 |

## 🔥 快速提升技巧

### 1. 开头 3 秒抓住注意力
```
❌ 今天我们来聊聊...
✅ 上周三，我做了个决定，现在想想都后怕。

❌ 很多人问...
✅ "你到底怎么坚持下来的？"老张问我。

❌ 在当今社会...
✅ 35 岁那年，我被裁员了。
```

### 2. 用对话代替叙述
```
❌ 他很生气地告诉我不要这样做。
✅ "别干了。"他把烟按灭，"这活儿不靠谱。"
```

### 3. 用动作代替形容
```
❌ 她很紧张。
✅ 她不停地转着笔，转掉了三次。

❌ 他很犹豫。
✅ 他手指在发送键上悬了十秒，最后还是关了窗口。
```

### 4. 制造金句
```
对比：不是...而是...
反转：你以为...其实...
数字：这 3 件事...
```

### 5. 结尾给行动
```
❌ 希望这篇文章对你有帮助。
✅ 今天就做一件事：____________________

❌ 欢迎大家留言讨论。
✅ 在评论区告诉我：____________________
```

## 📝 好文章结构模板

### 模板 A：故事 + 观点 + 行动
```
1. 讲一个具体故事（300-500 字）
2. 提炼一个观点（100-200 字）
3. 给一个行动建议（50-100 字）
```

### 模板 B：问题 + 分析 + 答案
```
1. 提出一个具体问题
2. 分析问题的原因
3. 给出解决方案
```

### 模板 C：清单式
```
1. 引入（为什么重要）
2. N 个要点（每点 + 例子）
3. 总结（一句话）
```

## 💡 写作心法

> **写你真正相信的东西。**

> **写给你想象中的一个人看，不是给"大家"看。**

> **写完大声读，不顺就改。**

> **好文章是改出来的，不是写出来的。**

> **持续写比写得好更重要。**

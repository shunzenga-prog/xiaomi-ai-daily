## 第一部分：细节描写练习

### 练习模板 1：五感观察法

**任务**：描写一个场景，必须包含至少 3 种感官

**模板**：
```
【视觉】我看到了...
【听觉】我听到了...
【嗅觉】我闻到了...
【味觉】我尝到了...
【触觉】我感受到了...
```

**范例**（描写早晨的厨房）：
```
阳光从百叶窗的缝隙里挤进来，在餐桌上切出几道亮斑。（视觉）
水壶在炉子上发出轻微的嘶嘶声，像有人在低声说话。（听觉）
咖啡粉的香气混着烤面包的焦香，暖烘烘地往鼻子里钻。（嗅觉）
我端起杯子，温热的瓷壁贴着掌心，一天的温度就这样开始了。（触觉）
```

**你的练习**：
选一个你熟悉的地方（办公室、卧室、常去的餐厅），用五感描写它。

---

### 练习模板 2：具体化练习

**任务**：把抽象的描述改成具体的画面

**模板**：
```
抽象：天气很好
具体：____________________

抽象：他很生气
具体：____________________

抽象：房间很乱
具体：____________________

抽象：她很漂亮
具体：____________________
```

**参考答案**：
```
抽象：天气很好
具体：阳光把梧桐叶的影子投在墙上，风一吹，影子就跟着晃

抽象：他很生气
具体：他把手机扣在桌上，屏幕朝下，一句话不说

抽象：房间很乱
具体：沙发上堆着三件外套，茶几上有两个喝了一半的杯子，
      一本翻开的书倒扣在地上

抽象：她很漂亮
具体：她笑起来的时候，眼睛先弯，然后嘴角才跟上，
      像有个开关被按下了
```

**要点**：
- 用动作代替形容词
- 用画面代替评价
- 用细节代替概括

## 第二部分：金句制造练习

### 练习模板 3：对比金句

**公式**：不是 A，而是 B

**范例**：
- 成熟不是学会表达，而是学会咽下
- 真正的自由不是想做什么就做什么，而是不想做什么就不做什么
- 好的爱情不是彼此凝视，而是朝同一个方向看

**你的练习**：
```
主题：成长
不是____________________，而是____________________

主题：工作
不是____________________，而是____________________

主题：爱情
不是____________________，而是____________________
```

**公式**：你以为 A，其实 B

**范例**：

- 你以为错过是遗憾，其实错过是保护
- 你以为他在忙，其实他只是不想理你

**你的练习**：
```
主题：努力

主题：选择
你以为____________________，其实____________________

主题：时间
你以为____________________，其实____________________
```

### 练习模板 5：数字金句

**公式**：用具体数字增加可信度

**范例**：
- 成年人的崩溃，往往从一条没回复的消息开始
- 一个人值不值得深交，看这 3 个细节就够了
- 真正厉害的人，都懂得在 3 件事上装傻

**你的练习**：
```
主题：习惯
____件事，决定了你的一天

主题：判断
看这____个细节，就知道____

主题：改变
____天后，你会感谢现在的自己
```

## 第三部分：个人声音练习

### 练习模板 6："我"的练习

**任务**：用第一人称写一段经历，必须包含：
- 一个具体的时间
- 一个具体的地点
- 一个具体的人
- 一句具体的对话
- 一个真实的感受

**范例**：
```
去年冬天，我在公司楼下的便利店遇到老张。
他穿着那件洗得发白的灰色羽绒服，手里拿着两罐啤酒。
"走，喝一杯？"他问我。
我看了看表，晚上九点半，明天还要早起。
但我说："好。"
有些时候，你需要一杯啤酒胜过需要八小时睡眠。
那天我们没聊工作，没聊房子，没聊孩子。
就聊当年一起追过的那个姑娘，现在过得怎么样。
```

**你的练习**：
回忆一个真实的场景，按上面的要求写下来。

### 练习模板 7：观点表达练习

**任务**：对一个有争议的话题表达你的真实看法

**模板**：
```
话题：____________________

大多数人说：____________________

但我觉得：____________________

因为：____________________

我经历过：____________________
```

**范例**：
```
话题：要不要合群

大多数人说：合群很重要，不合群会被孤立

但我觉得：合群没那么重要

因为：我见过太多为了合群而弄丢自己的人

我经历过：刚工作时拼命融入同事圈子，
          每天一起吃饭一起聊天，
          三年后才发现，那些人我一个都不记得了。
          真正记得的，是那些和我一样"不合群"的人。
```

## 第四部分：结构练习

### 练习模板 8：三段式结构

**公式**：故事 + 观点 + 行动

**模板**：
```
【故事】讲一个具体的故事（300-500 字）

【观点】从故事里提炼一个观点（100-200 字）

【行动】给读者一个具体的行动建议（50-100 字）
```

**范例**（主题：坚持）：
```
【故事】
我有个朋友想学画画，买了最好的画具，
报了最贵的班，结果坚持了两周就放弃了。
后来他换了个方式：每天只画 5 分钟。
就 5 分钟，画什么都行，画完就收起来。
这个习惯他保持了三年。
现在他的画不能说多好，但至少能看了。
最重要的是，他相信自己是个能坚持的人。

【观点】
很多时候，我们不是缺乏毅力，
而是把目标定得太高。
5 分钟听起来太简单，简单到不好意思放弃。
但正是这种"不好意思"，让我们坚持了下来。

【行动】
选一件你想坚持的事，把难度降到 5 分钟。
今天就做，明天继续。
```

**你的练习**：
选一个主题，用三段式结构写一篇文章。

### 练习模板 9：清单式结构

**公式**：主题 + N 个点 + 总结

**模板**：
```
【引入】为什么这个话题重要（100 字）

【清单】
1. 第一点 + 例子
2. 第二点 + 例子
3. 第三点 + 例子
...

【总结】一句话总结（50 字）
```

**范例**（主题：好朋友的特征）：
```
【引入】
人这一生会遇到很多人，但真正的朋友没几个。
怎么判断一个人值不值得深交？看这 4 点。

【清单】
1. 他记得你的小事
   你随口说想吃什么，下次见面他真带来了。

2. 他敢跟你说真话
   你错了，他不会附和，会告诉你。

3. 他不在你面前说别人坏话
   因为在你背后也不会说你的。

4. 他需要你的时候会开口
   不需要你的时候不打扰。

【总结】
好朋友不是天天联系，而是需要时永远在。
```

**你的练习**：
选一个主题，用清单式结构写一篇文章。

## 第五部分：修改练习

### 练习模板 10：删减练习

**任务**：把一段文字删掉 30%，但不损失信息

**原文**：
```
我觉得今天天气真的非常好，阳光明媚，
天空特别蓝，没有一点云彩，
让人心情也变得特别好，
我想出去走走，感受一下这美好的天气。
```

**修改后**：
```
今天天气很好，阳光把天空洗得发蓝。
我想出去走走。
```

**要点**：
- 删掉"我觉得""真的""特别"这类词
- 删掉重复的意思
- 用画面代替形容

**你的练习**：
找一段自己写的文字，删掉 30%。

---

### 练习模板 11：朗读修改法

**任务**：写完文章后大声读出来，修改不顺的地方

**检查清单**：
- [ ] 有没有拗口的句子？
- [ ] 有没有太长的句子？（超过 30 字要警惕）
- [ ] 有没有重复的词？
- [ ] 有没有假大空的词？（比如"非常重要""众所周知"）
- [ ] 读起来有节奏感吗？

**要点**：
- 不顺的地方就改
- 读着累的地方就删
- 卡壳的地方就拆

## 第六部分：30 天写作挑战

- Day 1：描写你的房间（五感）
- Day 2：描写一个陌生人（外貌 + 动作）
- Day 3：描写一顿饭（色香味）
- Day 4：描写一次通勤（所见所闻）
- Day 5：描写一个老物件（来历 + 故事）
- Day 6：描写一种天气（不用"好/坏"）
- Day 7：自由写作（30 分钟不间断）

### 第 2 周：金句训练
- Day 8：写 5 个"不是...而是..."金句
- Day 9：写 5 个"你以为...其实..."金句
- Day 10：写 5 个带数字的金句
- Day 11：选一个金句扩写成文章
- Day 12：收集 10 个你喜欢的金句，分析结构
- Day 13：模仿你喜欢的金句写 5 个
- Day 14：自由写作（30 分钟不间断）

### 第 3 周：个人声音训练
- Day 15：写一件让你后悔的事
- Day 16：写一个你感谢的人
- Day 17：写一个你坚持的习惯
- Day 18：写一个你改变的观点
- Day 19：写一个你学到的教训
- Day 20：写一个你的小癖好
- Day 21：自由写作（30 分钟不间断）

### 第 4 周：完整文章训练
- Day 22：三段式文章（故事 + 观点 + 行动）
- Day 23：清单式文章（N 个点）
- Day 24：问答式文章（自问自答）
- Day 25：对话式文章（记录一次对话）
- Day 26：评论式文章（评论一本书/电影）
- Day 27：模仿你喜欢的博主写一篇文章
- Day 28：自由写作（30 分钟不间断）
- Day 29：修改一篇之前的文章
- Day 30：总结这 30 天的收获

## 附录：写作检查清单

### 写前检查
- [ ] 我想表达什么核心观点？
- [ ] 我的读者是谁？他们关心什么？
- [ ] 我有什么个人经历可以用？
- [ ] 我想让读者读完做什么？

### 写后检查
- [ ] 开头能吸引人吗？
- [ ] 有具体的细节吗？
- [ ] 有金句吗？
- [ ] 有"我"在文章里吗？
- [ ] 读起来顺口吗？
- [ ] 对读者有价值吗？

### 发布前检查
- [ ] 标题够吸引人吗？
- [ ] 段落有呼吸感吗？
- [ ] 有错别字吗？
- [ ] 长度合适吗？（公众号文章 1500-3000 字最佳）

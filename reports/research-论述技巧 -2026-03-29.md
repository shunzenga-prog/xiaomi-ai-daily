## 一、"论述不足"具体指什么

根据 boss 反馈和写作规范分析，"论述不足"主要表现为：

| 问题类型 | 具体表现 |
|---------|---------|
| **事实支撑不足** | 只有观点陈述，缺少具体数据、案例、来源引用 |
| **分析深度不够** | 停留在表面描述，没有深入分析原因、影响、机制 |
| **论证逻辑松散** | 观点之间缺乏逻辑连接，没有形成完整的论证链条 |
| **缺少对比视角** | 没有正反对比、新旧对比、优劣对比来强化观点 |
| **实操性弱** | 只有"是什么"，缺少"怎么做"的具体步骤和方法 |

**核心问题**: 文章像"新闻摘要"而非"深度分析"——告诉了读者发生了什么，但没说明白为什么重要、怎么来的、意味着什么。

## 二、优秀科技文章的论述技巧

### 1. 经典论证结构（亚里士多德式）

```
提出问题 → 提出解决方案/观点 → 多维度论证 → 总结升华
```

- **问题**: 清晰定义要讨论的核心问题
- **方案**: 明确表达立场或解决方案
- **论证**: 用数据、案例、引用、对比等多种方式证明
- **升华**: 延伸到更大图景或行动建议

### 2. 分层递进结构

```
现象层（发生了什么）→ 原因层（为什么发生）→ 影响层（意味着什么）→ 行动层（我们该怎么做）
```

每层都要有事实支撑，避免跳跃。

### 3. 案例对比法

- **前后对比**: 技术/产品使用前后的效果差异
- **优劣对比**: 不同方案的优缺点分析
- **正反对比**: 成功案例 vs 失败案例

### 4. 数据论证法

- 引用权威数据来源（论文、报告、官方数据）
- 用具体数字代替模糊表述（"大幅提升"→"提升 37%"）
- 数据要有来源、有时间、有上下文

### 5. 引用背书法

- 引用领域专家观点
- 引用权威机构报告
- 引用一手资料（论文原文、官方公告）

## 三、3-5 个可操作的改进方法

### 方法 1: "三层论述法"（核心方法）

**每个重要观点必须包含三层**:

1. **观点层**: 清晰陈述观点
2. **支撑层**: 提供数据/案例/引用作为支撑
3. **解释层**: 解释支撑材料如何证明观点

**示例**:
```
❌ 论述不足：大模型正在快速发展

✅ 论述充分：
【观点】大模型正在快速发展
【支撑】2025 年 Q1 全球大模型参数量平均增长 3.2 倍，头部厂商迭代周期从 6 个月缩短至 2 个月（来源：AI Index Report 2025）
【解释】这意味着技术迭代速度已进入"月级"时代，跟不上节奏的厂商将在 3-6 个月内失去竞争力
```

### 方法 2: "5W1H 检查清单"

**每篇文章完成后对照检查**:

| 维度 | 检查问题 | 达标标准 |
|-----|---------|---------|
| What | 核心事件/技术说清楚了吗？ | 非专业人士能看懂 |
| Why | 为什么重要/会发生？ | 至少有 2 个原因分析 |
| Who | 涉及哪些关键角色？ | 主要参与者都有提及 |
| When | 时间线/节点清晰吗？ | 关键时间点明确 |
| Where | 应用场景/影响范围？ | 至少 1 个具体场景 |
| How | 怎么实现的/怎么做的？ | 有步骤或机制说明 |

**缺少任何一项=论述不足**

### 方法 3: "对比强化模板"

**在关键观点处强制加入对比**:

```markdown
### [观点标题]

**对比视角**:
- **之前**: [过去的情况/传统做法]
- **现在**: [当前的变化/新做法]
- **差异**: [核心变化点，用数据或具体描述]

**这意味着**: [对读者的实际影响]
```

### 方法 4: "来源标注规范"

**所有事实性陈述必须标注来源**:

```
❌ 据悉，很多公司都在用 AI

✅ 根据 2025 年 3 月 Gartner 报告，全球 67% 的企业已在生产环境部署 AI 应用
```

**来源优先级**: 一手资料 > 权威报告 > 主流媒体 > 行业分析 > 其他

### 方法 5: "深度追问法"

**对每个重要陈述连续追问 3 次**:

```
陈述：AI 正在改变内容创作

追问 1: 怎么改变的？→ 生成效率提升、成本降低
追问 2: 提升多少？→ 效率提升 5-10 倍，成本降低 70%
追问 3: 有什么证据？→ 某公司案例 + 行业报告数据
```

**追问不到第 3 层=论述深度不够**

---

## 四、快速自检流程

**文章发布前 5 分钟检查**:

1. [ ] 每个核心观点有数据/案例支撑吗？
2. [ ] 有至少 1 个对比分析吗？
3. [ ] 重要数据来源标注了吗？
4. [ ] 有"这意味着什么"的分析吗？
5. [ ] 有"具体怎么做"的建议吗？

**3 个以上❌ = 需要补充论述**

---

## 五、总结

**论述不足的本质**: 只有结论，没有论证过程。

**改进的核心**: 把"是什么"扩展成"是什么 + 为什么 + 怎么来的 + 意味着什么 + 怎么做"。

**最简单的方法**: 写完每个观点后，强制自己回答"你怎么知道？"和"那又怎样？"两个问题。

# 公众号排版技巧与视觉设计指南

> 本指南整理了微信公众号文章排版的最佳实践、常用元素规范和 AI 类文章的风格建议。

---

## 一、排版最佳实践

### 1. 整体原则

- **简洁清晰**：避免过度装饰，内容为主，样式为辅
- **层次分明**：通过字号、颜色、间距建立清晰的视觉层级
- **一致性**：全文保持统一的配色方案、字体风格和元素样式
- **移动端优先**：80% 以上用户在手机端阅读，确保小屏体验

### 2. 字号规范

| 内容类型 | 推荐字号 | 说明 |
|---------|---------|------|
| 正文 | 15-16px | 最舒适的阅读字号，14px 偏小，17px 偏大 |
| 一级标题 | 18-20px | 加粗，可配主题色 |
| 二级标题 | 17-18px | 加粗或加底色区分 |
| 三级标题 | 16px | 可加粗或与正文区分 |
| 注释/引用 | 14px | 配浅灰色 (#888888) |
| 图注 | 12-13px | 浅灰色，居中对齐 |

### 3. 行间距与段间距

- **行间距**：1.6-1.8 倍（公众号编辑器默认 1.75 倍最佳）
- **段间距**：空一行或使用 15-20px 段后距
- **两端缩进**：8-16px，避免文字贴边

### 4. 配色方案

**安全配色组合：**

| 类型 | 推荐颜色 | 用途 |
|-----|---------|------|
| 主色 | #0066FF / #1890FF / #07C160 | 标题、重点文字、链接 |
| 辅助色 | #FF6B6B / #FFA940 / #13C2C2 | 强调、标注、图表 |
| 正文色 | #333333 / #2C2C2C | 主要文字内容 |
| 次要色 | #666666 / #595959 | 次要信息、图注 |
| 背景色 | #F5F5F5 / #FAFAFA | 引用块、代码块背景 |

**配色原则：**
- 全文主色不超过 3 种
- 避免高饱和度颜色大面积使用
- 确保文字与背景对比度足够（WCAG AA 标准）

### 5. 页面宽度

- 公众号文章最佳阅读宽度：**600-640px**
- 图片宽度建议统一为 640px 或 1080px（Retina 屏）

## 二、常用排版元素规范

### 1. 标题样式

**一级标题（文章大标题）：**
```
【居中】
字号 20px，加粗
主题色或黑色
上下各留 20-24px 间距
可加下划线或底色装饰
```

**二级标题（章节标题）：**
```
【左对齐或居中】
字号 18px，加粗
可配左侧竖线/色块装饰
示例：
▌ 章节标题
或
━━ 章节标题 ━━
```

**三级标题（小节标题）：**
```
【左对齐】
字号 16px，加粗
可用 emoji 或符号引导
示例：
📌 小提示
✅ 关键点
```

### 2. 引用样式

**基础引用：**
```
左侧竖线 + 浅灰背景 + 浅灰文字
边框：3-4px  solid  主题色
背景：#F5F5F5 或 #F8F9FA
内边距：12-16px
文字：14-15px，#666666
```

**金句引用（居中突出）：**
```
【居中】
字号 16-17px，加粗
上下各加引号装饰「」
配浅色背景或渐变
```

### 3. 列表样式

**无序列表：**
```
使用统一符号：•  ○  ■  ●  ◆
符号与文字间距 8-10px
每项之间适当留白
```

**有序列表：**
```
数字加粗，配主题色
或使用带圈数字 ① ② ③
保持对齐一致
```

**检查清单：**
```
✅ 已完成项
⬜ 待办项
⚠️ 注意事项
```

### 4. 重点强调

**加粗强调：**
- 用于关键词、核心观点
- 不宜过多，每段 1-2 处即可

**高亮标记：**
```
背景色：#FFF3CD / #FFE58F
文字：保持黑色或深色
内边距：2-4px
```

**下划线：**
- 用于专业术语、首次出现的名词
- 避免与链接混淆

### 5. 代码块

```
等宽字体（Consolas、Monaco）
背景：#F6F8FA 或 #282C34（深色）
边框：1px solid #E1E4E8
内边距：12-16px
圆角：4-6px
支持语法高亮更佳
```

### 6. 图片规范

**尺寸：**
- 封面图：900×383px（2.35:1）
- 文中插图：640px 宽，高度自适应
- 长图：宽度 640px，单屏高度不超过 1200px

**格式：**
- 优先使用 WebP（更小体积）
- 兼容 JPG/PNG
- 单图不超过 5MB

**图注：**
```
【居中】
字号 12-13px
颜色：#999999
与图片间距 8px
```

### 7. 分割线

**简约款：**
```
━━━━━━━━━━━━━━
或
──────────────
```

**装饰款：**
```
✦ ━━━━━━━━━━━━━━ ✦
或
◇◆ ◇━━━━━━━ ◇◆ ◇
```

**颜色：** #E8E8E8 或主题色浅色版

### 8. 表格样式

```
边框：1px solid #E8E8E8
表头：背景 #FAFAFA，加粗
行高：40-48px
斑马纹：隔行浅灰背景
移动端：考虑横向滚动或简化
```

## 三、AI 类文章风格建议

### 1. 视觉调性

**科技感配色：**
- 主色：#0066FF（科技蓝）、#07C160（极客绿）、#722ED1（紫色）
- 辅助：#13C2C2（青色）、#FF6B6B（强调红）
- 背景：深色模式可用 #1A1A2E / #16213E

**风格关键词：**
- 简洁现代
- 专业可信
- 未来感但不浮夸

### 2. 内容结构

**推荐结构：**
```
1. 引子（痛点/场景/问题）
2. 核心观点（加粗突出）
3. 分点论述（小标题 + 案例）
4. 实操步骤（编号列表）
5. 总结回顾（引用框强调）
6. 互动引导（投票/留言）
```

### 3. 特殊元素

**技术术语标注：**
```
首次出现：加粗 + 简要解释
代码/命令：代码块样式
模型名称：统一格式如 **GPT-4** / **Claude 3**
```

**数据展示：**
```
关键数字：加大字号（24-32px）+ 主题色
对比数据：表格或简单图表
引用来源：小字标注于下方
```

**提示框（Tips）：**
```
💡 小贴士
背景：#E6F7FF
边框：1px solid #1890FF
内边距：12-16px
```

```
⚠️ 注意事项
背景：#FFF7E6
边框：1px solid #FA8C16
```

```
✅ 最佳实践
背景：#F6FFED
边框：1px solid #52C41A
```

### 4. 图表建议

**流程图：** 使用 Mermaid 或清晰截图
**对比图：** 左右/上下对比，标注清晰
**数据图：** 简洁柱状图/折线图，避免 3D 效果
**架构图：** 分层清晰，配色统一

### 5. 互动元素

**投票：** 文末插入公众号投票组件
**留言引导：** 
```
💬 你对这个话题怎么看？
欢迎在评论区留言讨论～
```

**推荐阅读：**
```
📚 延伸阅读
• [文章标题 1](链接)
• [文章标题 2](链接)
```

### 6. 开头与结尾模板

**开头：**
```
【场景引入】
你有没有遇到过...

【问题抛出】
今天我们来聊聊...

【价值承诺】
读完这篇，你将获得...
```

**结尾：**
```
【总结回顾】
以上就是...的核心要点

【行动号召】
如果对你有帮助，欢迎：
✓ 点赞 ✓ 在看 ✓ 分享给朋友

【下期预告】
下期我们将探讨...
```

## 四、排版工具推荐

### 在线编辑器
- **秀米**：模板丰富，适合新手
- **135 编辑器**：功能全面，素材多
- **i 排版**：简洁易用
- **Markdown Here**：Markdown 转公众号格式

### 设计工具
- **Canva**：封面图、插图设计
- **稿定设计**：公众号专用模板
- **Figma**：自定义排版组件

### 图片处理
- **TinyPNG**：图片压缩
- **Squoosh**：格式转换 + 压缩
- **Remove.bg**：一键抠图

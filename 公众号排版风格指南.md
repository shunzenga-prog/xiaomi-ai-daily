## 🎨 品牌色彩系统

### 主色调
| 颜色 | 色值 | 用途 |
|------|------|------|
| 小咪橙 | `#FF8C42` | 主标题、强调色 |
| 暖猫粉 | `#FFB6C1` | 副标题、装饰元素 |
| 云朵白 | `#FFF9F5` | 背景色 |
| 墨黑 | `#2D2D2D` | 正文文字 |
| 灰猫 | `#8B8B8B` | 辅助文字、引用 |

### 色彩使用原则
- 主色调占比不超过 30%
- 背景使用浅色，保证阅读舒适
- 强调色用于关键信息和行动号召

## 📝 标题样式

### 一级标题 (文章主标题)
```
格式：🐱 标题文字
样式：20px / 粗体 / 小咪橙 (#FF8C42)
对齐：居中
间距：上下各 24px
```

**示例：**
> 🐱 今天学会了新技能

### 二级标题 (章节标题)
```
格式：▌ 标题文字
样式：18px / 粗体 / 墨黑 (#2D2D2D)
左侧边框：4px 小咪橙
对齐：左对齐
间距：上 20px / 下 12px
```

**示例：**
> ▌ 第一部分：开始之前

### 三级标题 (小节标题)
```
格式：✦ 标题文字
样式：16px / 中粗体 / 暖猫粉 (#FFB6C1)
对齐：左对齐
间距：上 16px / 下 8px
```

**示例：**
> ✦ 小贴士

## 📄 正文排版

### 字体设置
| 元素 | 字体大小 | 行高 | 颜色 |
|------|----------|------|------|
| 正文 | 15px | 1.8 | 墨黑 #2D2D2D |
| 强调文字 | 15px | 1.8 | 小咪橙 #FF8C42 |
| 引用/备注 | 14px | 1.6 | 灰猫 #8B8B8B |
| 代码块 | 13px | 1.5 | 墨黑 #2D2D2D |

### 段落间距
- 段前：0px
- 段后：16px
- 首行缩进：无 (现代风格)

### 段落宽度
- 最大宽度：650px (保证移动端阅读舒适)
- 左右边距：16px

### 特殊元素样式

#### 引用块
```
样式：左侧 4px 暖猫粉边框
背景：云朵白 #FFF9F5
内边距：12px 16px
圆角：8px
```

#### 列表
- 无序列表：使用 🐾 作为项目符号
- 有序列表：使用 ① ② ③ 样式
- 列表项间距：8px

#### 代码块
```
背景：#F5F5F5
边框：1px solid #E0E0E0
圆角：6px
内边距：12px
字体：等宽字体
```

## 🎯 排版原则

### 1. 呼吸感
- 留白是设计的一部分
- 不要填满每一寸空间
- 让眼睛有休息的地方

### 2. 一致性
- 同级别标题样式统一
- 色彩使用有规律
- 全文字体不超过 3 种

### 3. 可读性
- 对比度足够 (文字与背景)
- 行高舒适 (1.6-1.8)
- 段落不要太长 (5-7 行最佳)

### 4. 品牌感
- 🐱 emoji 是核心识别元素
- 小咪橙是品牌色
- 温暖、亲切的语调
